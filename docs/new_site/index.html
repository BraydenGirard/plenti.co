New Site

Description

This is the first command you'll want to run for a new project. It creates the default folders and files for your website. 

Usage

You can create a new project by running plenti new site YOUR_SITE_NAME. 

Flags

LongShortDescription
--bare-b(Coming Soon) Creates a minimalist site scaffolding with less default content.
--theme="git@github.com:user/repo.git"-t="git@github.com:user/repo.git"(Coming Soon) Point to a Git repository to use a base theme for your site.
