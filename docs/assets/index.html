Assets

Static Files

Static files like images and PDFs can be stored in the top level assets/ directory. You can reference these assets from any template from the root of the site, for example: <img src="/assets/my-image.jpg" />. 

You can also use a path relative to the specific template. For instance if you were adding an image to layout/components/my-component.svelte, you could reference the same image like <img src="../../assets/my-image.jpg" /> if you'd like.
